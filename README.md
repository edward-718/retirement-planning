# 养老规划平台

一款帮助用户进行养老规划的 Web 应用，提供养老金测算、进度追踪、养老城市探索等功能。

## 功能特性

### 🏠 仪表盘
- 养老进度总览，目标金额、已储蓄、养老缺口、建议月存等核心数据一目了然
- 下一个里程碑追踪，实时掌握储蓄进度
- 存款充足度追踪，支持基础保障档、品质养老档、无忧尊享档三档切换
- 养老准备行动清单，将大目标拆解为可执行的小步骤
- 三支柱养老金概览（社保、企业年金、个人养老金）
- 每日养老知识小贴士

### 🧮 养老金测算器
- **极简模式**：约 30 秒完成 6 项输入，快速估算养老需求
- **标准模式**：约 2 分钟，包含社保、企业年金等详细信息，测算更精准
- 分步引导式输入，带表单校验
- 支持自动估算缺失数据

### 📋 养老蓝图
- 养老总缺口、建议每月储蓄、目标养老生活费三大核心指标
- 支持保守（2%）、中性（4%）、乐观（6%）三种收益情景切换
- 三支柱养老金饼图分析（月领取）
- 缺口构成分析柱状图
- 资金流向漏斗图
- 储蓄增长曲线（含目标线对比）
- AI 生成的专属规划建议

### 🎯 进度追踪
- **里程碑**：分阶段储蓄目标追踪，带进度条和状态标识
- **任务系统**：储蓄任务、知识任务、探索任务，完成可获得积分奖励
- **成就墙**：解锁式成就系统，激励持续行动
- 任务完成庆祝动效

### 🏙️ 养老城市探索
- 多城市养老推荐，支持按气候、生活成本、医疗资源、社交配套、交通便利、综合评分排序
- 城市五维评分雷达图（生活成本、医疗资源、气候舒适、社交配套、交通便利）
- 生活成本明细：简朴生活、舒适生活、品质生活三档展示
- 城市对比功能（最多 3 个城市）
- 城市收藏功能
- 养老成本 vs 舒适度散点分布图

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS
- **状态管理**：Zustand
- **路由管理**：React Router DOM
- **图表库**：Recharts
- **图标库**：Lucide React
- **工具库**：clsx、tailwind-merge

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```
